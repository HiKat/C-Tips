# C-Tips
### pointer01.c, pointer02.c
ポインタ変数の扱い. \*pと&pが表すもの.  
\*は「ポインタ型変数の指すメモリに入っている値を参照する」.    
&は「変数のアドレスを参照する」.

### argArray.c
関数に配列を引き渡す時の書き方.   

### copyArray01.c
配列のコピーの間違った方法. 配列の宣言での配列名のみはポインタ型であるがポインタ変数ではない.  
(配列の変数名だけを書けばそれを配列の先頭アドレスとみなすという言語仕様があるというだけ.)

### memcpy.c
配列のコピー  

### mainArg.c
main関数内のスコープの注意と、main関数内のローカル変数の書き換えのセオリー
